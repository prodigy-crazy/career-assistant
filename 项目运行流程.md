# 大学生成长职途助手 - 完整运行流程

## 项目概述

**项目名称：** 大学生成长职途助手  
**技术栈：** 纯原生 HTML + CSS + JavaScript（无任何第三方框架）  
**访问地址：** http://localhost:8000

---

## 页面结构

项目采用单页应用（SPA）架构，包含4个页面：

1. **首页（home）** - 欢迎页面和入口
2. **信息填写页（info）** - 用户基本信息收集
3. **职业测评页（test）** - 霍兰德职业兴趣测评
4. **结果展示页（result）** - 个性化职业规划报告

---

## 完整运行流程

### 第一步：访问首页

**URL：** http://localhost:8000

**页面内容：**
- 大标题："大学生成长职途助手"
- 副标题："按学期拆解目标，动态规划你的大学职途"
- 阶梯成长图形：展示大一→大二→大三→大四的成长路径
- 核心按钮："开始规划"
- 底部说明："基于霍兰德职业兴趣理论 · 原创测评体系"

**功能：**
- 点击"🌓"按钮切换暗黑/亮色主题
- 点击"开始规划"按钮进入信息填写页

---

### 第二步：填写信息

**页面标题：** 填写你的基础信息

**表单字段：**
1. **所在专业**（必填）
   - 单行输入框
   - 验证：不能为空

2. **当前年级**（必填）
   - 下拉选择框
   - 选项：大一、大二、大三、大四
   - 验证：必须选择一个选项

3. **已掌握技能**（选填）
   - 多行文本框
   - 提示语：可填写编程语言、软件、特长等

4. **意向方向**（选填）
   - 多选框
   - 选项：就业、考研、考公、暂未确定

**交互逻辑：**
1. 点击"← 返回"回到首页
2. 点击"下一步，去测评"：
   - 先校验必填项
   - 校验失败显示错误提示
   - 校验成功将数据存入 `localStorage`，然后跳转到测评页

**数据存储：**
```javascript
localStorage.setItem('userInfo', JSON.stringify({
    major: '计算机科学与技术',
    grade: 'sophomore',
    skills: 'Python, Java',
    directions: ['job', 'graduate']
}));
```

---

### 第三步：职业测评

**页面标题：** 霍兰德职业兴趣测评

**测评规则：**
- 总共20道题
- 分6个维度：
  - R（现实型）- 4题
  - I（研究型）- 4题
  - A（艺术型）- 3题
  - S（社会型）- 3题
  - E（企业型）- 3题
  - C（常规型）- 3题

**每题选项（5分制）：**
- 非常不符合 = 1分
- 不符合 = 2分
- 一般 = 3分
- 符合 = 4分
- 非常符合 = 5分

**交互流程：**
1. **顶部进度条**
   - 显示："第 X 题 / 共 20 题"
   - 实时更新答题进度

2. **题目区域**
   - 显示当前题目编号和题干
   - 5个选项按钮横向/竖向排列
   - 选中选项有高亮效果

3. **底部导航**
   - "上一题"按钮（第1题隐藏）
   - "下一题"按钮（第20题显示"提交测评"）

4. **进度保存**
   - 每次答题自动保存到 `localStorage`
   - 刷新页面不丢失已答题目

5. **提交计算**
   - 点击"提交测评"后
   - 自动计算每个维度的总分
   - 将得分存入 `localStorage`
   - 跳转到结果展示页

**数据存储：**
```javascript
// 答题进度
localStorage.setItem('testAnswers', JSON.stringify({
    1: 4,
    2: 3,
    3: 5,
    // ... 共20题
}));

// 各维度得分
localStorage.setItem('testScores', JSON.stringify({
    R: 16,  // 现实型
    I: 18,  // 研究型
    A: 12,  // 艺术型
    S: 14,  // 社会型
    E: 15,  // 企业型
    C: 13   // 常规型
}));
```

---

### 第四步：查看结果报告

**页面标题：** 你的专属职途规划报告

**模块1：兴趣维度雷达图**
- 使用Canvas绘制六边形雷达图
- 6个顶点对应R/I/A/S/E/C六个维度
- 图形用主色（#4A90E2）半透明填充
- 标注每个维度的实际得分

**模块2：推荐发展路线**
- 根据得分最高的3个维度
- 结合用户的专业、年级、意向方向
- 生成3条适配的发展路线：

  1. **技术就业路线**
     - 适配度：根据R、I维度得分计算
     - 描述：适合喜欢动手实践、追求技术深度的同学

  2. **考研深造路线**
     - 适配度：根据I、A维度得分计算
     - 描述：适合喜欢理论研究、追求学术发展的同学

  3. **公考体制路线**
     - 适配度：根据S、E、C维度得分计算
     - 描述：适合注重稳定、擅长沟通协调的同学

**模块3：能力差距分析**
- 结合推荐的第一条路线
- 列出3项需要重点提升的能力
- 每项包含简短说明和提升方向

**模块4：本学期行动清单**
- 根据用户年级生成5条任务
- 按优先级排序（1-5）
- 每条任务可勾选完成状态
- 打卡状态自动保存到 `localStorage`

**年级任务示例：**
- 大一：参加新生专业导学活动、学习基础编程语言等
- 大二：参加专业技能竞赛、完成技术小项目等
- 大三：争取名企实习、系统准备秋招/春招等
- 大四：全力冲刺秋招/春招、完成毕业设计等

**底部操作按钮：**
1. **导出报告** - 生成Markdown格式报告文件
2. **生成海报** - 生成PNG格式图片
3. **重新测评** - 清空数据，回到首页

## 技术实现要点

### 页面导航

```javascript
function nav(pageId) {
    // 移除所有页面的active类
    document.querySelectorAll('.page').forEach(page => {
        page.classList.remove('active');
    });
    // 显示目标页面
    document.getElementById(pageId).classList.add('active');
    
    // 如果是结果页，重新生成任务列表
    if (pageId === 'result') {
        loadResult();
    }
}
```

### 雷达图绘制

使用Canvas 2D API绘制六边形雷达图：
- 计算6个顶点的坐标
- 绘制同心六边形网格
- 绘制从中心到各顶点的轴线
- 根据得分绘制数据区域
- 添加维度标签

### 响应式设计

- 移动端适配（375px）
- 桌面端适配（1200px最大宽度）
- CSS媒体查询处理不同屏幕尺寸
